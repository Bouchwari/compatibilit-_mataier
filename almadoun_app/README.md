# نظام إدارة المستهلكات — الثانوية الإعدادية ألمدون

## المتطلبات
- Python 3.8 أو أحدث
- نظام Windows / Linux / macOS

## التثبيت والتشغيل

### Windows
انقر نقراً مزدوجاً على الملف:
```
تشغيل_التطبيق.bat
```

### Linux / macOS
```bash
pip install -r requirements.txt
python main.py
```
